# 入力信号
- opcode[3..0]: オペコードの値
- dataA[15..0]: 第1オペランドの値
- dataB[15..0]: 第2オペランドの値

# 出力信号
- out[15..0]: dataA (opcodeに対応する演算) dataBの結果の値
- cond[3..0]: 条件コード...
